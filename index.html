ETCH
Englishtown Core Helper

Contains the small but most important utilities which elab uses.
etch should be placed into the <head /> to make sure it is loaded first to provide the very basic functionalities which makes our web page look great.
because of the <script /> usually blocking the page from loading the other resources, it is very important to keep it small and only essential utilities can be put into etch.

ETUI
Englishtown User Interface

Optional UI lib, provides the ui components such as font sprite, scrollbar, lightbox and so on.
